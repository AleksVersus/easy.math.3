История версий

История версий ветки 3.x.x совпадает с историей ветки 2.x.x до версии 2.4.0.

  • Версия 3.0.0:

    1. Первая стабильная версия новой ветки.
    2. Переработан мануал, возможно с ошибками.
    3. Написан скрипт, конвертирующий офлайн-версию мануала в онлайн.

  • Версия 3.1.0:

    1. Нигде в мануале не указано, что это библиотека для QSP-плеера. Исправил на стартовой странице.
    2. Теперь офлайн мануал собирается из онлайн-версии страниц.
    3. Добавлена функция em.hex.add — складывает два шестнадцатеричных числа.
    4. Добавлена функция em.arr.fill — упрощённое заполнение массива значениями (жертвуем скоростью в пользу удобства записи).
    5. Исправлена функция em.sqrt — неверный аргумент.
    6. Добавлена функция em.gen_uuid — генератор UUID.
    7. Добавлена функция em.str.slice - вырезает из строки подстроку при указании символов
    8. Ускорена работа функции em.byte.add минимум на треть. В некоторых случаях работает вдвое быстрее предыдущего варианта. Так же теперь, если функция принимает кривое значение в виде аргумента, она возвращает строку '[not a number]'
    9. Добавлены функции работы с HTML.
    10. Вынесены в отдельный файл переменные, определяющие, какие функции будут добавлены в сборку.
