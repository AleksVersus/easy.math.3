Работа с HTML

em.html.clr_pref

Очищает html-разметку от преформатирования.

Функция использует цикл для избавления от символов преформатирования, что может существенно подвешивать игру на больших фрагментах HTML-разметки. Используйте рационально.

Аргументы:
$args[0] - текст, в котором присутствует разметка.
Пример вызова функции:
$html = {
<qhtml>
  <table border=1>
    <tr>
      <td width=25% bgcolor=#ffffee>
        Некий текст
      </td>
      <td>Ещё текст</td>
    </tr>
    <tr>
      <td width=25% bgcolor=#ffffee>
        Вторая строка
      </td>
      <td>12345677 < ></td>
    </tr>
  </table>
</qhtml>
}

@em.html.clr_pref($html)
Результат работы функции:

(подсветка для наглядности)

<qhtml><table border=1><tr><td width=25% bgcolor=#ffffee>Некий текст</td><td>Ещё текст</td></tr><tr><td width=25% bgcolor=#ffffee>Вторая строка</td><td>12345677< ></td></tr></table></qhtml>

em.html.to_mnemonics

Замена в тексте с html-разметкой символов < и > на мнемоники &lt; и &gt; соответственно.

Бывает полезно, когда включен режим распознавания HTML, но при этом нужно вывести текст с разметкой как есть.

Аргументы:
$args[0] - текст, в котором присутствует разметка.
Пример вызова функции:

(подсветка для наглядности)

$html = {<span style="color:red;">Подсвеченный красным <em>итальянский курсив</em></span>}

@em.html.to_mnemonics($html)
Результат работы функции:
&lt;span style="color:red;"&gt;Подсвеченный красным &lt;em&gt;итальянский курсив&lt;/em&gt;&lt;/span&gt;
